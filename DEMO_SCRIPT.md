# 产业链图谱分析平台 — 5 分钟演示脚本

## 开场（30 秒）

大家好，今天给大家演示我们的 **A 股产业链图谱分析平台**。

这是一个基于龙虎榜、市场行情和财经新闻数据，构建产业链关系图谱并进行可视化分析的工具。平台采用纯静态架构，数据采集和前端渲染完全分离，可以一键部署到 GitHub Pages。

**技术栈：** Python 后端（akshare + SQLite + pandas）| 原生 JS 前端（ECharts）| GitHub Actions CI/CD

先访问 `http://127.0.0.1:8030/app/index.html`，打开平台主页，展示顶部 Tab 栏和整体 UI。

> **操作：** 启动开发服务器，浏览器打开页面，逐个指向八个 Tab 名称，让观众对平台全貌有初步印象。

---

## 第一幕：龙虎榜数据采集与入库（40 秒）

> **操作：** 切换到终端窗口，执行数据同步命令。

```bash
PYTHONPATH=src python3 -m stockgraph.cli.sync_dragon_tiger --date 2026-04-17
```

**旁白要点：**

- 平台自动从东方财富 API 抓取龙虎榜数据，包含上榜个股、营业部席位、买卖金额
- 数据源采用多级降级策略：东方财富 API → 东方财富网页 → 新浪 → 同花顺 → StockAPI
- 入库前自动识别席位类型：**机构席位**、**外资席位**、**游资席位**（章盟主、赵老哥等知名游资自动标记）
- 数据存入 SQLite，支持增量更新和历史回溯

> **操作：** 命令执行完成后，简要展示终端输出的成功日志。

---

## 第二幕：龙虎榜查询（50 秒）

> **操作：** 切换回浏览器，点击第一个 Tab「龙虎榜查询」。

**旁白要点：**

- 左侧日期选择器可以切换交易日
- 首页自动展示当日 **活跃个股 TOP** 和 **活跃席位 TOP** 排行榜

> **操作：** 展示活跃个股和活跃席位表格，点击某只股票名称。

**旁白要点：**

- 点击个股后，右侧展示该股票当日所有席位操作明细
- 买方席位用红色标记，卖方席位用绿色标记，金额一目了然
- 支持按股票代码、席位名称进行关键词搜索

> **操作：** 在搜索框输入一只股票代码（如 600519），展示搜索结果。

---

## 第三幕：龙虎榜关系图谱（50 秒）

> **操作：** 点击第二个 Tab「龙虎榜关系网」。

**旁白要点：**

- 这是平台的核心功能 —— **产业链关系图谱**
- 节点分为两类：**营业部席位**（菱形）和 **个股**（圆形）
- 边代表席位对个股的买卖操作，边的粗细反映交易金额
- 颜色编码：**红色 = 机构**，**蓝色 = 游资**，**绿色 = 外资**

> **操作：** 展示力导向图的动态布局效果，鼠标悬停某个节点，高亮相邻节点和边。

**旁白要点：**

- 通过这个图谱，我们可以直观看到：
  - 哪些游资在协同作战（共同出现在同一批股票上）
  - 某只股票背后有哪些资金力量在博弈
  - 机构和游资对同一只股票的分歧

> **操作：** 在关键词过滤框输入一个游资名称（如「章盟主」），展示过滤后的子图。

**旁白要点：**

- 过滤后只保留相关子图，帮助聚焦分析特定资金的操盘路径

---

## 第四幕：市场热度 3D 可视化（40 秒）

> **操作：** 点击第三个 Tab「热度图」。

**旁白要点：**

- 这是一个 **3D 散点图**，将全市场个股映射到三维空间
- X/Y/Z 轴和点的大小都可以自由配置，比如：
  - X 轴 = 涨跌幅，Y 轴 = 关注度排名，Z 轴 = 讨论排名
  - 点的大小 = 交易热度排名

> **操作：** 展示 3D 图表的旋转、缩放交互，切换不同的维度配置。

**旁白要点：**

- 支持按交易所筛选（沪主板、深主板、创业板、科创板、北交所）
- 支持按行业板块筛选
- 支持关键词搜索特定个股
- 底部联动数据表格，鼠标悬停 3D 点可以看到个股详情

> **操作：** 鼠标悬停某个散点，展示 tooltip 信息。

---

## 第五幕：行业轮动分析（40 秒）

> **操作：** 点击第四个 Tab「全 A 图谱」。

**旁白要点：**

- 这是基于 ECharts Graph 的全市场大图谱，股票、行业、交易所、交易状态、席位、新闻、事件和情绪都可以作为节点
- 每只股票都会连接到所属行业、交易所、涨跌状态和成交额分层
- 龙虎榜数据会形成席位到股票的资金边，新闻数据会形成新闻到股票、事件和情绪的语义边

> **操作：** 切换日期、行业、交易所和节点类型，搜索某只股票或某个行业，点击节点查看右侧强关联关系。

---

## 第六幕：行业轮动分析（40 秒）

> **操作：** 点击第五个 Tab「行业日历」。

**旁白要点：**

- 以日历形式展示每日领涨行业，颜色区分不同行业板块
- 可以快速识别行业轮动规律：某个行业连续多日领涨，说明主线行情正在形成

> **操作：** 展示日历视图，点击某一天的色块，查看当日详细数据。

> **操作：** 点击第六个 Tab「行业强弱」。

**旁白要点：**

- 左侧展示领涨行业 TOP 排行，右侧展示领跌行业 TOP 排行
- 采用水平柱状图，直观对比各行业的强弱差异
- 底部提供完整的行业排名表格

> **操作：** 展示图表和表格，让观众看到行业分布的全貌。

---

## 第七幕：个股新闻与情感分析（40 秒）

> **操作：** 点击第七个 Tab「个股新闻」。

**旁白要点：**

- 平台自动抓取个股新闻，进行 **情感分析** 和 **事件分类**
- 情感分为正面（红色标签）、负面（绿色标签）、中性（灰色标签）
- 事件类型包括：业绩、并购重组、监管、投融资、产业政策

> **操作：** 展示股票卡片网格，点击某只股票卡片。

**旁白要点：**

- 点击后进入新闻详情时间线，展示该股票的所有相关新闻
- 每条新闻显示标题、来源、发布时间、情感标签、事件类型
- 可以按情感类型过滤新闻

> **操作：** 切换情感过滤按钮，展示不同情感下的新闻筛选结果。

---

## 第八幕：AI 智能分析（30 秒）

> **操作：** 点击第八个 Tab「AI 分析」。

**旁白要点：**

- 平台集成了 OpenAI 兼容的 LLM API，可以对个股进行 AI 智能分析
- 选择一只股票后，系统自动聚合该股票的 **新闻数据** 和 **龙虎榜数据** 作为上下文
- 支持自定义分析提示词

> **操作：** 选择一只股票，点击「开始分析」，展示流式输出效果。

**旁白要点：**

- AI 会综合新闻情感、龙虎榜资金动向、行业趋势等信息，给出投资分析建议
- 采用 SSE 流式输出，实时展示分析过程

---

## 第九幕：工程架构与部署（20 秒）

**旁白要点：**

- **数据层：** SQLite 单文件数据库，16 张表，18 个索引，支持百万级数据
- **采集层：** 多数据源降级策略 + 自适应提供商评分机制
- **分析层：** 网络图谱构建（二部图、席位共现图、个股共现图）
- **展示层：** 纯静态 SPA，每个 Tab 独立加载数据，互不影响
- **部署：** GitHub Actions 自动生成静态站点并推送到 GitHub Pages；push 只做离线校验，手动或定时触发才抓取远端数据

```bash
# 一键部署
bash run_daily_sync.sh   # cron 定时任务：采集 → 分析 → 构建 → 部署
```
